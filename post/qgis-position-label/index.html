Choisir la position des labels dans QGIS

Cet article a été testé pour QGIS Wien 2.8

QGIS positionne automatiquement ces labels, et offre différentes options pour modifier sa méthode: on peut par exemple forcer un décalage automatique du texte depuis le point d’ancrage. Cependant cette modification ne peut pas se faire label par label. La méthode suivante permet de positionner le texte des labels.

1- Ajouter des colonnes à la table des attributs.

Clique droit sur la couche à modifier, puis “Open Attributes Table”. Pour ajouter une colonne, passer en mode édition et cliquer sur l’icône ajouter.

S’il est impossible de passer en mode édition, celà vient surement du fait que la couche est en geojson ou csv, il suffit de la convertir en shapefile: clique droit sur la couche, save As, sélectionner Shapefile. Copier coller le style si besoin puis supprimer la couche initiale.

Trois colonnes à ajouter:

  • x: contient l’abscisse du label
    • name: x
    • type: Decimal number (real)
    • Width: Nombre de chiffres: 20 est bien suffisant
    • Precision: Nombre de chiffres après la virgule: 10 est bien suffisant
  • y: contient l’ordonnée du label
    • name: x
    • type: Decimal number (real)
    • Width: 20
    • Precision: 10

Enregistrer les changements.

2- Faire correspondre les nouvelles colonnes x et y aux coordonnées des labels

Aller dans le menu des labels >> placement, et faire correspondre la coordonnée X du label à la colonne x de la couche, et Y à y.

Sélectionner x dans le menu déroulant qui apparaît, option “Attribu Field >> Field Type”

3- Positionner les labels

Une fois les deux étapes faites, passer la couche en mode édition, le bouton pour bouger les labels est actif, cliquer dessus et positionner les labels à leur place.
